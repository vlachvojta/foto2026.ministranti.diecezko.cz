# Diecézko 2026 - fotografie

Po přidání nového roku využij fork na minulé repo. Ve forku změň tyto věci:
- index.html 
- v terminálu běž do složky scripts/
- spusť `..upload-images.sh <source_folder> <destination_folder>`
- gallery.json
  - aktualizuj nastavení v horní části
  - spusť pomocí `yarn gallery`
- zkontroluj, commit, push
- zkopíruj gallery.json na hlavní stránky ministranti.diecezko.cz
